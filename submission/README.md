# Submission Package

This folder is the final-project evidence package. It is intentionally split so required local evidence is not confused with optional public-IP bonus evidence.

## Folder Contract

- `local/`: required evidence for the 12 email items from `docs/final-project.txt`, items `a` through `l`.
- `public/`: optional public-IP bonus evidence only. Public evidence does not replace any required local item.
- `email/`: final email draft plus a pointer to the consolidated submission file.
- `archive/pre-2026-06-12-structure/`: previous package layout and stale copied evidence preserved for traceability.

## Readiness Rules

- `ready`: file exists and matches the assignment item.
- `partial`: useful evidence exists, but another required artifact or manual check is still missing.
- `missing`: no acceptable evidence is packaged yet.
- `manual`: the evidence requires user action, a browser screenshot, external UI capture, or a user-run Gatling flow.

Do not send the package until `docs/submission.md` shows every required local item as ready or explicitly calls out the remaining manual check.
