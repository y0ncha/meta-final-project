# MTA DevOps Final Project

## Project Purpose

This repository contains the JSP app, containerized Tomcat deployment, Jenkins CI/CD flow, Playwright validation, Gatling performance tests, HAR evidence, and submission artifacts for the MTA 2026 Semester B DevOps final project.

## Local URLs

- Tomcat application: `http://localhost:8080/meta/`
- Jenkins: `http://localhost:8081/`

The `meta` context path is defined by the Maven WAR final name in `docs/plans/03-jsp-maven-war-app.md`.

## Repository Status

- Public GitHub repository: not configured yet

## Evidence Policy

Generated evidence is saved under ignored `output/` paths and attached to the final submission package instead of committed by default. Review HAR files, screenshots, Playwright reports, Gatling reports, PDFs, Jenkins logs, and monitoring exports before sharing them.
